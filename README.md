# 🌍 Earthquake Visualization Project

A Python project that visualizes global earthquake data using the USGS API and Plotly.

---

## 🚀 Features
- Fetches real-time earthquake data (last 24 hours)
- Processes GeoJSON data
- Visualizes earthquakes on a world map
- Color-coded by magnitude
- Interactive hover information

---

## 🧠 Technologies Used
- Python
- Requests
- Plotly
- USGS Earthquake API

---

## 📊 Visualization
- Marker size = magnitude
- Marker color = intensity
- Hover shows place, magnitude, depth
